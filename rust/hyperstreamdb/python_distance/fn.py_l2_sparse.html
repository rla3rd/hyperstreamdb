Compute L2 distance between two sparse vectors

This function computes L2 (Euclidean) distance efficiently for sparse vectors by only processing non-zero elements. The result is mathematically equivalent to converting to dense and computing L2 distance, but much more efficient for high-dimensional sparse data.

§Parameters

a : SparseVector First sparse vector b : SparseVector Second sparse vector device : ComputeContext, optional Optional ComputeContext for GPU acceleration (currently unused for sparse)

§Returns

float L2 distance between sparse vectors

§Raises

§Examples

import hyperstreamdb as hdb a = hdb.SparseVector([0, 5], [1.0, 2.0], 10) b = hdb.SparseVector([0, 3], [1.0, 3.0], 10) distance = hdb.l2_sparse(a, b) print(f“{distance:.3f}“) 3.606
